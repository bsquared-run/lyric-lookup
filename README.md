# Lyric Lookup

This project is a web application that allows you to look up song lyrics. It uses a Next.js frontend and a Python backend. This project is open source and contributions are welcome.

## Prerequisites

Before you begin, ensure you have the following installed:
*   [Node.js](https://nodejs.org/) (which includes npm)
*   [Python](https://www.python.org/)

## Getting Started

To get the application running, follow these steps:

### 1. Clone the Repository

```bash
git clone https://github.com/bsquared-run/lyric-lookup.git
cd lyric-lookup
```

### 2. Set Up and Run the Backend

In the project's root directory, run the following command. This will create a Python virtual environment, install the necessary dependencies, and start the backend server.

```bash
run_backend.bat
```

### 3. Set Up and Run the Frontend

In a separate terminal, also in the project's root directory, run this command to install the frontend dependencies and start the Next.js development server:

```bash
run.bat
```

### 4. View the Application

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
